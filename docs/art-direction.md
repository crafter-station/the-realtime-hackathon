# Art Direction — grounded in Lusion.co

> Reference: **https://lusion.co** (Awwwards-winning 3D/interactive studio).
> This replaces the "orange particles in black space" direction, which read as
> generic AI-3D. The goal: Lusion's craft language, adapted to Portal's brand and
> the hackathon, with the **portal-traversal-on-scroll** concept the client asked for.

## 1. Core aesthetic (what Lusion actually is)

- **Style:** *Editorial 3D craft.* A clean, bright, type-led editorial page that
  **frames** beautifully-rendered, tactile 3D — not a full-screen abstract void.
- **Philosophy:** the page is calm and confident; the 3D is a contained spectacle
  with **real materials and light**, so it feels made, not generated.
- **Not this:** dark space, glowing particle blobs, wormhole clichés, neon haze.

## 2. Color palette

Lusion runs **light**, which is the single biggest correction.

| Role | Lusion | Ours (Portal-branded) | Hex |
| --- | --- | --- | --- |
| Page background | off-white / cool lavender | warm off-white | `#F4F2EE` / `#F0F1FA` |
| Ink (type) | near-black | near-black | `#0A0A0A` |
| Accent / signature | electric blue + cyan ribbon | **Portal orange** | `#FF4D00` |
| Scene card | near-black rounded panel | near-black | `#0E0E10` |
| Secondary ink | grey | grey | `#8A8A8A` |

Total: ~5. Light page + black type + **one hot orange** for the ribbon, the portal
glow, and highlights. The dark comes only *inside* the 3D scene / the portal's far side.

## 3. Typography system

- **Display:** large **neo-grotesque** (Lusion uses **Aeonik**, weight ~400–500),
  set HUGE — the type *is* the hero. Tight tracking, generous leading, left-aligned,
  editorial. e.g. "Bold Ideas, Brought to Life" at ~10–14vw.
- **Body:** same grotesque, ~18–22px, set in a narrow right-hand column beside the
  headline (two-column editorial), grey-ink, comfortable line-height.
- **Decision for us:** the current brand font is *Geist Pixel* (pixel) — that clashes
  with this clean editorial look. **Proposal: use Geist Sans** (the non-pixel Geist —
  a free Aeonik-adjacent grotesque) for all display/body, and keep the **pixel font
  only for tiny mono labels/eyebrows** (e.g. `SCROLL TO EXPLORE`, timestamps) so the
  hackathon keeps a thread of its identity. This is the one call I need you to confirm.

## 4. Key design elements

- **Contained 3D:** the scene lives inside a **rounded-corner card** (dark) on the
  light page — or fills the viewport at signature moments. Real materials: glossy /
  matte plastic, subsurface, studio lighting. Tactile physical objects, or **real
  cinematic footage** (Lusion floats an astronaut) inside device/monitor frames.
- **Signature ribbon:** a single **glossy flowing tube** (Lusion's cyan) weaving
  across sections — ours in **orange**, catching light. One memorable through-line.
- **Chrome:** pill buttons (`LET'S TALK •`, `MENU ••`), a circular `—` control,
  `+` corner ticks, a thin `SCROLL TO EXPLORE` cue. Restrained, precise.
- **Layout:** big type left / body column right, huge whitespace, a real grid,
  rounded cards. Calm rhythm; the 3D provides the drama.

## 5. The portal concept (the client's explicit ask)

> "Entras al portal de la hackathon → al hacer scroll pasas al OTRO LADO del portal
> → sigues avanzando → llegas a otra sección."

Concretely, as a scroll-driven sequence:

1. **Arrival (light, editorial):** hero on the off-white page — huge type
   *"The Realtime Hackathon"*, body column, and a **portal** rendered inside a
   rounded dark card: a glossy orange **ring with a real lens-flare**, calm, inviting.
2. **Approach:** scrolling dollies the camera **toward** the portal; the card grows
   to fill the viewport (the frame dissolves), lens-flare blooms.
3. **Through:** you **pass to the other side** — a short, cinematic threshold moment
   (light → the portal's interior). No hyperspace cliché; a clean push-through with
   depth-of-field and a bloom pulse.
4. **The other side (a built world):** you emerge into a distinct, tactile
   environment and **keep advancing** — this is where Portal's realtime capability is
   shown as one crafted scene (not five particle blobs). Then the page returns to the
   light editorial rhythm for prizes / format / register.

Contrast is the point: **calm light editorial ↔ the dark, luminous inside of the
portal.** Passing between them is the signature moment.

## 6. What makes it *not* slop

- Light editorial page + bold grotesque type doing the heavy lifting (not the 3D).
- Real materials and lighting; contained, framed 3D; one signature object.
- A single strong idea (pass through the portal) executed with restraint.
- Craft details: rounded cards, pill chrome, `+` ticks, precise grid, whitespace.
